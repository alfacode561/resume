Project Overview

A WebRTC-based video chat application enabling real-time communication between multiple participants. Features include screen sharing, text chat, and room creation capabilities.

Key Features

  • Multi-participant video calls
  • Screen sharing
  • Real-time chat
  • Room creation and joining
  • Low-latency communication
  • End-to-end encryption
  • Virtual backgrounds
  • Recording capabilities

Node.js WebRTC Socket.io React WebAssembly Redis
